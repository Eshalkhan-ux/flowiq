💸 FlowIQ — Smart Cash Flow Tracker

FlowIQ is a modern, AI-powered cash flow management web application built for the Micathon’26 “Money Moves” theme. It enables individuals and small businesses to track, analyze, and predict their financial movement in real-time, turning complex data into clear, actionable insights.

live demo: flowiq-two.vercel.app

🚀 Core Features
💼 Smart Onboarding
Enter cash-in-hand and upload bank statements (PDF/Image) to automatically calculate your total opening balance, reducing manual effort and errors.
⚡ Quick Log System
Add transactions using natural language input like “paid rent 40000” or “received 85000 from Ali” for fast and intuitive tracking.
📊 Dashboard Overview
Monitor live balance, monthly income, and expenses with real-time updates in a clean, modern UI.
🧾 Transaction Management
Easily log and categorize transactions (salary, rent, utilities, etc.) for structured financial tracking.
🤖 Intelligent Features
🔹 Real-Time AI Categorization

The onDescType() engine actively assists users while entering transaction details:

Detects keywords like rent, salary, WAPDA, FBR to auto-assign category and type
Displays real-time AI hints to guide users
Ensures high accuracy with minimal user effort
